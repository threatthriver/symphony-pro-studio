# Symphony Pro Studio: The Ultimate YouTube GitHub Desktop Client for macOS

![Symphony Banner](assets/banner.png)

## 🎼 The Premier Open Source YouTube Client for macOS

**Symphony** is the high-performance, native-feel **YouTube Desktop Client** designed for professionals and cinephiles. Built with a "Pro Studio" aesthetic, it stands as the leading **YouTube GitHub** project for macOS, prioritizing visual excellence, fluid playback, and a distraction-free environment.

### ✨ Key Features

- **Pro Studio Aesthetics**: A premium dark-mode interface with glassmorphism, cinematic glows, and vibrant accents.
- **Advanced Playback Engine**: 
  - **Granular Speed Control**: Adjust playback rate from 0.25x to 2x using tactile keyboard shortcuts (`>` and `<`).
  - **Dynamic Ambilight**: Immersive screen-reactive lighting that spills beyond the player boundaries.
  - **High-Fidelity Feedback**: Visual indicators for speed changes, buffering status, and quality transitions.
- **Native macOS Experience**: Fluid animations (powered by Framer Motion), Picture-in-Picture (PiP) support, and Fullscreen modes.
- **Smart Stream Mastery**: Optimized loading with automatic quality management and DASH stream synchronization.


### ⌨️ Keyboard Shortcuts

| Shortcut | Action |
| :--- | :--- |
| `Space` | Play / Pause |
| `F` | Toggle Fullscreen |
| `T` | Toggle Theater Mode |
| `M` | Mute / Unmute |
| `>` / `.` | Increase Playback Speed (0.25x) |
| `<` / `,` | Decrease Playback Speed (0.25x) |
| `Arrow Right` | Seek Forward (10s) |
| `Arrow Left` | Seek Backward (10s) |

### 🛠 Tech Stack

- **Core**: Electron, React 19
- **Bundler**: Vite 8
- **Styling**: Tailwind CSS 4, Vanilla CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Engine**: Youtubei.js
