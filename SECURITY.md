# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes              |
| < 1.0.0 | ❌ No               |

## Reporting a Vulnerability

If you discover a security vulnerability within Symphony, please send an e-mail to Aniket Kumar. All security vulnerabilities will be promptly addressed.

We request that you do not report security vulnerabilities through public GitHub issues.
